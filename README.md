# speakify

speakify is a speech to text program

## Requirements

- Linux / Windows / Mac

- Python 3+

- pip3

- SpeechRecognition

## Installation

`git clone https://github.com/iinc0gnit0/speakify && cd speakify && chmod +x * && sudo ./install.sh`

## Usage

You can use this commmand from any path: `speakify`
