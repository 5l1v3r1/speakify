# speakify

speakify is a speech to text program

## Requirements

- Linux

- Python 3+

- pip3

- SpeechRecognition

## Installation

`git clone https://github.com/iinc0gnit0/speakify && cd speakify && chmod +x * && sudo ./install.sh`

## Usage

`python3 speakify.py`
